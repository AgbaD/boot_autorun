# boot_autorun
A script for running scripts automatically after boot

## usage
   - Windows 
     - run script
       ```
       >py tool.py
       ```
     - Enter username 
     - Enter file path
       ```
       > C:\{username}\Desktop\{script.py}
       ```
   - Linux
     - run script
       ```
       $python3 tool.py
       ```
     - Enter file path
       ```
       > C:\{username}\Desktop\{script.py}
       ```
