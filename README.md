# boot_autorun
A script for running scripts automatically after boot

## itch
for windows, it might not work if you have whitespace in your username

## usage
   - Windows 
     - run script
       ```
       >py tool.py
       ```
     - Enter file path
       ```
       > C:\{username}\Desktop\{script.py}
       ```
   - Linux
     - run script
       ```
       $python3 tool.py
       ```
     - Enter file path
       ```
       > C:\{username}\Desktop\{script.py}
       ```
